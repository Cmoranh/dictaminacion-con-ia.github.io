Dictaminación con IA · sube un HTML y publícalo como nueva versión

1 Token de GitHub

Créalo en github.com/settings/personal-access-tokens/new: Repository access → Only select repositories → dictaminacion-con-ia.github.io · Permissions → Contents → Read and write. El token nunca sale de tu navegador; solo se envía a api.github.com.

2 Archivo HTML

Arrastra aquí el archivo HTML
o haz clic para seleccionarlo

3 Versión y despliegue

Se actualizará la leyenda "v# Piloto" del título y encabezado a la versión indicada, se hará commit a index.html y GitHub Pages lo publicará automáticamente (~1 min). Cada despliegue queda como commit, reversible desde el historial.
